# web

> A Vue.js project

## Build Setup

``` bash
# install dependencies
# 首次进入项目包管理使用
npm install

# serve with hot reload at localhost:8080
# 本地开发使用
npm run dev

# build for production with minification
# 发布打包使用
npm run build

# build for production and view the bundle analyzer report
# 打包且分析包文件结构使用
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```
